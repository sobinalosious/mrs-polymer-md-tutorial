# MRS Polymer MD Tutorial

This repository contains a Binder-ready workshop on extracting polymer properties from molecular dynamics outputs.

The tutorial focuses on:

- glass transition temperature
- dielectric constant (dipole component)
- thermal conductivity
- specific heat capacity

## Start Here

Open the workshop materials in:

- `MRS_TUTORIAL/notebooks/00_Start_Here.ipynb`

## Binder Launch

Launch Binder with:

```text
https://mybinder.org/v2/gh/sobinalosious/mrs-polymer-md-tutorial/main?urlpath=lab/tree/MRS_TUTORIAL/notebooks/00_Start_Here.ipynb
```
